# mememox (🧵 メモツール)

Markdownファイルをベースとした、カード型のブロック構造を持つ軽量・安全なエントリー型メモシステムです。
PCブラウザおよびスマートフォンで快適に動作し、データをローカルフォルダまたはサーバーへ隔離して安全に管理できます。

## ディレクトリ構造

* `app/` - フロントエンドの静的Webアプリケーション
  * `index.html` - メインのHTMLファイル
  * `style.css` - UIレイアウトとデザイン
  * `app.js` - アプリの制御・状態管理ロジック
  * `src/` - 各種機能モジュール
    * `storage.js` - メモデータ（デモ、ローカル、サーバーAPI）の入出力
    * `ui.js` - DOMレンダリングおよびキーボードショートカット
    * `markdown.js` - 独自Markdown形式のパースとシリアライズ
    * `utils.js` - 共通ユーティリティ
  * `how-to-use.md` - アプリケーションの操作マニュアル
* `util/` - システム連携等に用いる予備の空ディレクトリ（Git追跡対象外）
* `specification_for_api.md` - サーバーサイドで動作するAPI（PHP等）を導入・生成するためのセキュリティガイドライン
* `specification.md` - アプリ全体の基本設計・動作仕様書

## 起動方法

### ローカル（デモモード / ローカルフォルダ）
1. `app/index.html` をブラウザで直接開きます。
2. 「デモモード」で動作を開始します。
3. 画面右上の **フォルダを開く** ボタン（📁）からローカルの任意のフォルダを選択することで、ブラウザの File System Access API を介してローカルのMarkdownメモを直接読み書きできるようになります。

### サーバー連携モード
データをサーバーの公開領域外に隔離して、複数端末から安全にAPI経由でアクセスさせることができます。
詳細は、ルートディレクトリ内の [specification_for_api.md](specification_for_api.md) をご参照ください。

## ライセンス・ポリシー

* セキュリティ向上の観点から、サーバーサイドAPIの実装スクリプト（`api.php` 等）は配布していません。
* [specification_for_api.md](specification_for_api.md) の内容を生成系AI（ChatGPT, Claude, Gemini等）に入力し、お使いのホスティングサーバーに合わせたAPIスクリプトを生成して設置してください。
