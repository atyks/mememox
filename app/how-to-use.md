# entry-memo

`entry-memo` は、ブラウザ上で個人用Markdownメモを効率的に管理・閲覧・編集するためのデスクトップ向けキーボードフレンドリーなメモアプリケーションです。

時系列でログが流れて結論が分からなくなるチャットツールとは異なり、エントリーの上部に常に「概要（方針や結論）」を固定し、下部に「ブロック（時系列のメモ）」をカード状に蓄積・整理することができます。

---

## 🚀 起動方法

本アプリはWebサーバーの起動や特別な環境構築を必要とせず、ダウンロードしてすぐに利用可能です。

1. [index.html](file:///app/app/index.html) をダブルクリックします。
2. Microsoft Edge または Google Chrome などの Chromium 系ブラウザで開きます。
3. 最初は画面左上が「デモモード」になっています。まずはこの状態で各種操作をお試しください。
4. 実際のローカルファイルを保存・管理する場合は、画面左上の **[📁 メモフォルダを開く]** ボタンを押し、`entry-memo` 用のルートフォルダ（空のフォルダでも可）を選択します。
   * ※ブラウザのセキュリティ機能により、OS of システムフォルダや隠しフォルダが含まれる場所は選択できません。通常のドキュメントフォルダ等をご指定ください。

---

## 📁 実ファイル利用時の想定構成

ローカルフォルダを選択して使用する場合、以下のようなフォルダ構造が自動生成（または読み込み）されます。

```text
選択したルートフォルダ/
├─ inbox/
│  └─ inbox.md             (一時的なメモエントリー)
├─ thinking/
│  ├─ entry-memo-system.md
│  └─ pkm-and-markdown.md  (思考整理用のエントリー)
├─ work/
│  └─ work-memo-environment.md (作業ログ用エントリー)
└─ ゴミ箱/                  (削除されたエントリーの移動先)
```

## ✨ 主な機能

### 📌 概要セクション
* 各エントリーの最上部には、現在の結論、未解決事項、次にやるべきことを記述する「概要」を固定表示します。
* テキストエリアは入力された文字の量に合わせて**自動的に高さが伸び縮み**します。

### 📝 ブロックカード一覧
* 各ブロックは、見出し単位で個別のカードとして並びます。
* 追加したブロックには自動的に一意のID（例: `[a73bc]`）と追加時のタイムスタンプが付与されます。
* 本文のテキストエリアも、入力した文字数に応じてリアルタイムに高さが自動調整されます。

### 🗑️ エントリーの削除と「ゴミ箱」カテゴリー
* エントリーを削除すると、そのエントリーは物理削除されずに **`ゴミ箱`** という名前のカテゴリー（`ゴミ箱/` フォルダ）へ安全に移動（リロケート）されます。
* すでに `ゴミ箱` カテゴリーにあるエントリーを削除しようとした場合のみ、確認を経てPC内から完全に削除されます。

### ✏️ エントリーの編集（カテゴリー移動・リネーム）
* エントリータイトルを編集すると、Markdownファイル内のH1が書き換わるとともに、ファイル名もタイトルに合わせて自動でリネームされます。
* 所属するカテゴリーを変更して別のカテゴリーにエントリーを移動させたり、その場で「新しいカテゴリー」を作成してそこへ移動させたりできます。

### 🤝 ブロックのマージ機能
* 複数のブロック（チェックボックスで選択）を1つに統合して、別の既存エントリーや新規エントリーに切り出すことができます。
* マージする際、元の各ブロックの見出しレベルは自動的に1段下がり（デモートされ）、マージ先の書き込みが100%成功した後にのみ、元のエントリーから選択されたブロックが削除される安全設計です。

## ⌨️ キーボードショートカット

画面上で `?` キー (Shift + `/`) を押すと、いつでも右側からショートカットヘルプパネルを呼び出すことができます。

### A. アプリナビゲーション
* `?` : ヘルプパネルの開閉
* `Ctrl + \` : 左サイドバーの表示/非表示（トグル）
* `Alt + ↑` / `↓` : 前後のエントリーに切り替え
* `Alt + ←` / `→` : 「エントリー詳細ビュー」と「カテゴリーのエントリー一覧ビュー」の切り替え
* `Ctrl + O` : メモフォルダを開く

### B. 基本操作
* `Ctrl + C` : 新規エントリーの作成
  * ※ 画面上のテキストを選択している状態、または入力中の場合は、通常の「文字コピー」が優先されます。
* `Ctrl + E` : 現在開いているエントリーの編集（タイトル・カテゴリーの変更）
* `Shift + D` / `Delete` : エントリーの削除（ゴミ箱へ移動、または完全削除）
* `Ctrl + Enter` : ブロック追加ポップアップを開く
* `Ctrl + I` : 概要セクションの直接編集を開始
* `Ctrl + H` : 現在のエントリーのあるカテゴリーのエントリー一覧を表示
* `Ctrl + J` / `K` : カテゴリーの切り替え（前後のカテゴリーのエントリー一覧に移動）

### C. 項目選択時（J/K での移動時）
* `J` / `K` : 次 / 前の項目を選択（最後 ↔ 最初でループ）
* `Space` : 選択中のブロックカードの全文表示 / 折りたたみ
* `Enter` :
  * ブロックカード選択時 ──> 選択中のカードを編集
  * 概要選択時 ──> 概要セクションの編集を開始
* `M` : 選択中のブロックカードを別のエントリーへ移動
* `D` : 選択中のブロックカードを削除
* `X` : マージ対象として選択 / 解除
* `Ctrl + M` : 選択したブロックのマージを直接実行

### D. ポップアップ（モーダル）内
* `Ctrl + Enter` : 保存して閉じる
* `Esc` : 保存せずに閉じる

## ⚠️ 既知の制約
* Chromium系ブラウザ（Microsoft Edge, Google Chrome）での動作を想定しています。
* フォルダの自動保存（復元）はブラウザのセキュリティ制限やアクセス権のクリア状況により、ページ再読み込み時に再度フォルダ選択が必要となる場合があります。
* 規定のMarkdown形式（H1, ## 概要, ## ブロック, H3[ID] 等）に従っていないエントリーファイルは「編集不可」となります。手動でテキストエディタ等で修正してください。
