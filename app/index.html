    ナビゲーション

    ? ヘルプパネルの開閉
    Ctrl + \ サイドバーの表示/非表示
    Alt + ↑/↓ 前後のエントリーに切り替え
    Alt + ←/→ ビューの切り替え
    Ctrl + O メモフォルダを開く

    基本操作

    Ctrl + C 新規エントリーの作成
    Ctrl + E エントリーの編集
    Shift + D / Delete エントリーの削除 (ゴミ箱へ移動)
    Ctrl + Enter ブロック追加ポップアップを開く
    Ctrl + I 概要セクションの編集
    Ctrl + H カテゴリーのエントリー一覧を表示
    Ctrl + J / K カテゴリーの切り替え / エントリー一覧へ

    カテゴリーのエントリー一覧画面時

    J / K 次 / 前のエントリーを選択
    Enter 選択中のエントリーを開く
    Ctrl + J / K 前後のカテゴリーのエントリー一覧に移動

    概要・ブロックカード選択時（J/Kで移動）

    J / K 次 / 前の項目（概要・ブロック）を選択
    Space 選択中のカードの開閉
    Enter / E 選択中の項目（概要・ブロック）を編集
    M 選択中のカードを移動
    D 選択中のカードを削除
    X マージ対象として選択 / 解除
    Ctrl + M 選択したブロックのマージを実行

    ポップアップ（モーダル）内

    Ctrl + Enter 保存して閉じる
    Esc 保存せずに閉じる
